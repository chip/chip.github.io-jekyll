Man page structure

The basics

  • NAME - describes a command or system function
  • SYNOPSIS - basic usage of a command
  • DESCRIPTION - expanded explanation of the command
  • OPTIONS - explains various flags that the command supports

A few others

  • ENVIRONMENT
  • HISTORY
  • BUGS
  • AUTHOR
  • COPYRIGHT
  • EXAMPLES
  • DIAGNOSTICS
  • SEE ALSO
